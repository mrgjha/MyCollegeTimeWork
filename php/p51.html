PHP Versus C++




    

  • Variables begin with $ sign ($name = “John Doe”;)

  • No explicit declaration of variable types 

  • Introduction of “lazy” functions (foreach, explode, mail)

  • No Function Overloading

  • “Hidden” functions-within-a-function

  • Compiled during every page load*

  • Managed Language, no need to delete an object

  • Documented!  http://php.net/function_name (Ex: www.php.net/explode)


  • Web Specific:

  • Cookies and “Sessions”

  • Dynamic HTML based on user-defined logic

  • Interact and process a form’s action

  • Process URL Parameters (Ex: google.com/search.php&q=SomeValue)

  • Easy Database Integration (Which I will talk about)

  • Cross-Site-Scripting (XSS) security hacks
