Visual Studio.Net

  • Visual Studio.Net is a graphical interactive development environment (IDE).
  • It supports VB.Net development for the MS .Net 1.1 Framework – a set of DLLs that contain Microsoft’s latest development platform. This Framework (DLLs) can be installed on older operating systems like Windows 98 & Windows ME.
  • VB.Net cannot run on those OS’s, but the programs you develop with VB.Net can run on those platforms if the .Net 1.1 Framework is installed.
  • The .Net 1.1 Framework can be downloaded here:
  • .NET Framework Version 1.1 Redistributable Package http://msdn.microsoft.com/downloads/
