How to set the path for Windows 2000/NT/XP


1. Go to the Start menu, point to Settings, 
and select the Control Panel.
2. Edit the environment variables for your system. 
For NT, select the System icon, and the
Environment tab. For 2000, select the System icon, 
the Advanced tab, and the Environment
Variables button. 
For XP, 
select the Performance and Maintenance icon, 
the System icon, the Advanced tab, 
and the Environment Variables button.
3. Add c:\j2sdk1.4.0\bin to the far right side 
of the current path in System Variables and
select OK.

4.For CLASSPATH 
click New
Enter Variable Name  CLASSPATH
Enter Variable Value c:\j2sdk1.4.0\bin

Then
Restart

How to check the current path and classpath


Start a DOS prompt and enter the set command. 


When you do, the DOS window will
display various configuration data including the path 
and classpath.
