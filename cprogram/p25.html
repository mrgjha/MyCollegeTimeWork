Maybe try to use this feature to make your source codes more readable 
and not to mess them up.
Like in C, variables can be encapsulated between { } hooks. 
Then they are local to the zone encapsulated between the { and }.
Whatever happens with such variables inside the encapsulated zone 
will have no effect outside the zone: 



using namespace std;
#include<iostream>

int main ()
{   

double a;   
cout << "Type a number: ";   
cin >> a;   
{      
int a = 1;      
a = a * 10 + 4;      
cout << "Local number: " << a << endl;   
}   
cout << "You typed: " << a << endl;   
return 0;

}
