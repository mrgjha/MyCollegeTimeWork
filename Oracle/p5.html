Functional Dependence

  • For attribute sets: X and Y
    • X = {x1,x2, ...., xm}
    • Y = {y1,y2, ...., yn}
    • X ––> Y
    • "X functionally determines Y"
    • "Y functionally dependent on X"
    • if and only if each value of X has associated with it precisely one Y value

Name, dept_no, and dept_name are functionally dependent on emp_no. (emp_no -> name, dept_no,dept_name)
Skills is not functionally dependent on emp_no since it is not unique to each emp_no.

Transitive Dependence

Transitive dependency is a functional dependency between nonkey attributes
Dept_no and dept_name are functionally dependent on emp_no
however, department can be considered a separate entity.
