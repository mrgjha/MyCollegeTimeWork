Visual Studio.Net

Syntax:
For index=start To end [Step step]
     [statements]
     [Exit For]
     [statements]
 Next [index]
E.g.
Module Module1
  Sub Main()
      Dim loopIndex As Integer
      For loopIndex=0 to 3 
          System.Console.WriteLine(“Hello!”)
       Next loopIndex
   End Sub
End Module
