Visual Studio.Net

  1. Start Visual Studio.Net 2003
  2. Click on New Project Button
  3. In left window, select “Visual Basic Projects”.
  4. In right window, select “Console Application”
  5. Name it HelloWorld (Note: No Spaces!)
  6. Click “Browse” button to select your M:\CS161 Folder.
  7. Note location where project will be saved.
  8. Click “OK” button to create the project.

    HelloWorld VB.Net

  9. Add the following lines to the code window
  10. between Sub End Sub 
     Sub Main()
 Console.WriteLine("Hello World!!!")
 Console.ReadLine()
 End Sub
  11. Next, on the toolbar, click the triangle next to the Debug box, which will start your program.
  12. Note the console window.
  13. Press enter to exist that window.
